# Trainova

A train booking API built with ASP.NET Core, designed using Clean Architecture principles. The project explores backend engineering concepts such as CQRS, authentication, caching, and reliable booking workflows with concurrency control.

## Tech Stack

- **Framework**: ASP.NET Core (.NET 10)
